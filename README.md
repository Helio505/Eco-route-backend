# Backend do Projeto Hackathon 2024 - EcoRoute

## Tecnologias

- Node.js
- TypeScript
- NestJS
- Prisma
- PostgreSQL
- Docker

## Executar o projeto

1. Verificar se pré-requisitos estão instalados.

- Node.js
- Docker
- Docker Compose

2. Subir containeres do Docker.

```bash
docker-compose up --build
```
